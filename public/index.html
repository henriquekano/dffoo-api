Default parameters

  • _limit=20 or _limit={_start} + 20 or _end={_start} + 20 Pagination will be limited to 20 entities by default. Max is 50.

Custom query parameters

  • {field_name}_ninc=Value For array fields. Stands for "not includes". Only supports single values
  • {field_name}_inc=Value For array fields. Stands for "includes". Only supports single values
  • {field_name}_distinct Returns all the distincts values of 'field_name' across all entities
